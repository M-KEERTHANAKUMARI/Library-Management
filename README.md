Project Description

The Online Book Management System is a Java-based web application developed using Servlets, JDBC, and HTML.
It allows an administrator to add books, store them in a database, and search/view book details using the ISBN number.

This project demonstrates:

MVC architecture

Server-side validation

Database connectivity using JDBC

Session handling using HTTP sessions
